How to use this database
------------------------

Step 1 - Download Aldo's latest version of  Bruteforce Save Data and Install it. [http://ps3tools.aldostools.org/BruteforceSaveData.rar]

Step 2 - When you launch BSD, it will prompt a "Update Cheats Repository" window, if you have a old db.

Step 3 - Simply, click the "Download" button from the "Update Cheats Repository" window.

Step 4 - BSD will automatically download the latest version of the entire db to C:\Program Files\Bruteforce Save Data\Cheats\

Enjoy.
